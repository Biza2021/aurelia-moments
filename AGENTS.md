# Aurelia Moments Project Rules

- Preserve the luxury Aurelia Moments brand identity.
- Keep the public website mobile-first and close to the Stitch design.
- Use provided images from `public/images/aurelia`.
- Do not use placeholder images if real assets exist.
- Keep admin routes secure.
- Use TypeScript and clean reusable components.
- Do not add unrelated services.
- Keep copy elegant, modest, and suitable for Qatar.
- Do not turn this into an e-commerce store.
